How to Use the Translation App

1. Input Text and Languages

On the home page of the Translation App, you will find a form with fields to input the text you want to translate, the source language, and the target language.

Fill in the respective fields:

  • Text to Translate: Enter the text you want to translate.
  • Source Language: Specify the source language (e.g., "en" for English).
  • Target Language: Specify the target language (e.g., "fr" for French).

2. Click Translate

Once you've entered the required information, click the "Translate" button. The app will send a request to the Translation API to process the translation.

3. View Translation Result

After processing, the translated text will be displayed on the same page. You'll see the original input and the corresponding translated text.
